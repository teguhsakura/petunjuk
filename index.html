Panduan Penggunaan Alat

1
Pemasangan Alat Monitoring

Tancapkan alat monitoring ke port yang sesuai:

  • Untuk Alat Monitoring MPPT di tancapkan pada Port 485 ke SCC MPPT merk Techfine, Samoto, atau Amsan Hijau (60A).
  • Untuk Alat Monitoring Inverter Hybrid di tancapkan pada RS232 (Bukan Port BMS) ke Inverter Hybrid merk Techfine, Amsan, atau Samoto.
2
Koneksi WiFi Awal

Buka pengaturan WiFi di HP Anda, cari SSID WiFi bernama Teguh Sakura dan lalu hubungkan ke situ (tanpa password).

3
Konfigurasi WiFi dan Buka Web Dashboard

Halaman web dashboard untuk pengaturan akan terbuka otomatis (Sign-in WiFi). Jika tidak, buka web browser dan akses IP alat di http://192.168.4.1/

  • Klik menu WiFi Settings.
  • Tunggu proses pindai, pilih Nama WiFi yang ada di tempat Anda.
  • Masukkan Password WiFi nya dan klik Simpan.
  • Setelah sukses, Salin Link Alamat IP dengan klik Salin.
  • Setelah itu buka Web Browser, lalu dengan klik lama pada kolom URL agar keluar menu Paste, lalu Paste agar Alamat IP di masukkan ke Kolom URL lalu Enter, untuk membuka Halaman Web Dashboard.
  • Lalu scroll ke bawah dan klik Info Akun Blynk untuk mendapatkan informasi User dan Password buat Log In ke Aplikasi Blynk.
4
Download & Instalasi Blynk Legacy

Aplikasi ini memerlukan Blynk Legacy (Blynk Versi lama). Silahkan download melalui link di bawah:

Download Versi Arm64-v8A Download Versi Armeabi-v7A

Jika versi Arm64-v8A tidak bisa di install atau jalankan, maka download versi Armeabi-v7A.



Lalu jalanan Aplikasi Blynk nya, sebelumnya perlu di seting pakai Custom Server ( Cek tutorialnya dibawah )
